## 特性

* [x] **Monadic** A Parser is a monad.
* [x] **Declarative** Declarative grammar definition
* [x] **Operator**  Operator-based combinators(`<<`, `>>`, `&`, `|`, `/`, `@`)
* [x] **Lazy Evalution** Lazy evaluation for recursion
* [x] **Curried** Curried functional interfaces
